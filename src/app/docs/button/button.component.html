Base-Button

Use Bootstrap’s custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

Examples

Bootstrap includes several predefined button styles, each serving its own semantic purpose, with a few extras thrown in for more control.

primary secondary info success danger warning light dark
Code 
// pass color as Input
<base-button color="primary" >primary </base-button>
<base-button color="secondary" >secondary </base-button>
<base-button color="info" >info </base-button>
<base-button color="success" >success </base-button>
<base-button color="danger" >danger </base-button>
<base-button color="warning" >warning </base-button>
<base-button color="light" >light </base-button>
<base-button color="dark" >dark </base-button>

Outline Buttons

In need of a button, but not the hefty background colors they bring? Add the Input outline ones to remove all background images and colors on any button.

primary secondary info success danger warning light dark
Code 
// pass color as Input
<base-button color="outline-primary" >primary </base-button>
<base-button color="outline-secondary" >secondary </base-button>
<base-button color="outline-info" >info </base-button>
<base-button color="outline-success" >success </base-button>
<base-button color="outline-danger" >danger </base-button>
<base-button color="outline-warning" >warning </base-button>
<base-button color="outline-light" >light </base-button>
<base-button color="outline-dark" >dark </base-button>

Pill Button

Add Input pill to show button as pill.

primary secondary info success danger warning light dark
Code 
// pass color as Input
<base-button color="outline-primary" type="pill">primary </base-button>
<base-button color="outline-secondary" type="pill">secondary </base-button>
<base-button color="outline-info" type="pill">info </base-button>
<base-button color="outline-success" type="pill">success </base-button>
<base-button color="danger" type="pill">danger </base-button>
<base-button color="warning" type="pill">warning </base-button>
<base-button color="light" type="pill">light </base-button>
<base-button color="dark" type="pill">dark </base-button>

Size

Fancy larger or smaller buttons? Add Input sm or lg for additional sizes.

smaller medium larger
Code 
// pass color & size as Input
<base-button color="primary" size="sm" >smaller </base-button>
<base-button color="secondary" >medium </base-button>
<base-button color="warning" size="lg" >larger </base-button>

Block Level

Create block level buttons—those that span the full width of a parent—by adding Input display - block

block
Code 
// pass color & display as Input
<base-button color="secondary" display="block" >smaller </base-button>
