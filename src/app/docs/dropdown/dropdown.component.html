Base-Dropdown

Toggle contextual overlays for displaying lists of links and more with the Bootstrap dropdown plugin.

Examples

base-dropdown Login Register
Code 
// pass align  as Input in base-dropdown-menu
<base-dropdown>
    <a class="nav-link" data-toggle="dropdown" >base-dropdown</a>
    <base-dropdown-menu >
        <a class="dropdown-item" >Login</a>
        <a class="dropdown-item" >Register</a>    
    </base-dropdown-menu>
</base-dropdown>
