Popovers

Documentation and examples for adding Bootstrap popovers, like those found in iOS, to any element on your site.

Code 
 // It's pure bootstrap
<button type="button" class="btn btn-secondary mr-1" data-container="body" data-toggle="popover" data-placement="top" data-content="Vivamus sagittis lacus vel augue laoreet rutrum faucibus.">
    Popover on top
</button>

    Popover on right
</button>
